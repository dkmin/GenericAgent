## 한 줄로

- **vs Claude Code / Codex CLI**: 같은 "에이전트형 CLI" 스타일이지만, GenericAgent는 **모델 락인이 없고 세션 간 진화한다.** Skill이 누적된다는 점이 결정적 차이.
- **vs AutoGPT**: AutoGPT가 "task-specific 도구를 많이 가진" 방식이라면, GenericAgent는 "원자 도구 9개를 합성하는" 방식. 결과: 코드 규모 1/10, 환각 1/N.
- **vs Manus**: Manus는 SaaS — 사용자가 자기 환경/로그인을 양도해야 함. GenericAgent는 **로컬 우선** — 당신 PC의 로그인 세션에 직접 주입.
- **vs OpenClaw**: 530K줄 vs 3K줄. OpenClaw는 풀스택 솔루션, GenericAgent는 시드 → 자기진화.

## 어떤 사람에게 GenericAgent가 맞나

| 만약... | GenericAgent가 적합 | 다른 게 적합 |
|---|---|---|
| 내 PC의 브라우저 로그인을 그대로 쓰고 싶다 | ✅ | Manus ❌ |
| 모델을 자유롭게 갈아끼우고 싶다 (Claude→Gemini→Kimi) | ✅ | Claude Code ❌ |
| 오래 쓸수록 똑똑해지는 에이전트를 원한다 | ✅ | Codex/AutoGPT ❌ |
| 코드 베이스 전체를 1시간 안에 이해하고 싶다 | ✅ (3K줄) | OpenClaw ❌ |
| 오늘 당장 ChatOps만 필요하다 | ⚠️ (셋업 필요) | Manus/Claude Code ✅ |
