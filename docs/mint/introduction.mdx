**GenericAgent**는 약 3,000줄짜리 자가진화형 자율 에이전트 프레임워크입니다. 9개의 atomic tool과 100줄 남짓한 Agent Loop만으로 LLM이 로컬 컴퓨터를 시스템 레벨에서 제어합니다 — 브라우저, 터미널, 파일시스템, 키보드/마우스, 화면, 모바일(ADB)까지.

> **설계 철학**: 스킬을 미리 심지 마라. 진화시켜라.

새 작업을 풀 때마다 GenericAgent는 실행 경로를 SOP로 결정체화해서 메모리에 적습니다. 오래 쓸수록 **당신만의 스킬 트리**가 자라납니다.

## 다른 에이전트와 무엇이 다른가?

| 항목 | GenericAgent | 일반적인 에이전트 |
|---|---|---|
| 코드 베이스 | **~3K 줄** | 수만 줄 |
| 컨텍스트 윈도우 | **`<30K`** | 200K~1M |
| 스킬 획득 | 사용 중 자동 결정체화 | 사전 주입 필요 |
| 새 도메인 진입 | atomic tool 조합 + 메모리 | 새 도구 추가 |
