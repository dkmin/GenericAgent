## 언제 쓰는가
GenericAgent의 메모리에 없는 능력이 필요한데, 어떤 도구/SOP가 존재하는지 모를 때. 105K+ 스킬 카드에서 시맨틱 검색.

## 작동 원리 (Why)
혼자서는 모든 도구·라이브러리를 알 수 없습니다. `skill_search`는 외부 등록된 105K+ 스킬 카드 인덱스에 시맨틱 쿼리를 던지고 가장 부합하는 것들을 점수와 함께 반환.

## 가장 간단한 호출

```python
import sys
sys.path.append('../memory/skill_search')
from skill_search import search

# ⚠️ 반드시 영어 쿼리 — 한국어/중국어 매칭 효과 매우 낮음
results = search("python send email")
for r in results:
    s = r.skill
    print(f"[{r.final_score:.2f}] {s.name} — {s.one_line_summary}")
    print(f"  key: {s.key}  category: {s.category}  tags: {s.tags[:3]}")
```

## API 시그니처

```python
search(query, env=None, category=None, top_k=10) -> list[SearchResult]
#  env: 자동 감지, 보통 미지정
#  category: 선택 필터 (예: "devops")
#  top_k: 반환 수, 기본 10
```

## 반환 구조

```
SearchResult
  .final_score    float       종합 점수 (0~1)
  .relevance      float       시맨틱 관련도
  .quality        float       품질 점수
  .match_reasons  list[str]   매칭 이유
  .warnings       list[str]   경고
  .skill          SkillIndex ↓

SkillIndex (자주 쓰는 필드)
  .key                str       고유 식별자/경로
  .name               str
  .one_line_summary   str
  .description        str
  .category           str
  .tags               list[str]
  .form               str       sop/script/...
  .autonomous_safe    bool      자율 실행 안전 여부
```

## CLI

```bash
python -m skill_search "python testing"
python -m skill_search "docker deployment" --category devops --top 5
python -m skill_search "git" --json
python -m skill_search --stats
python -m skill_search --env
```

## 설정

| 항목 | 기본값 | 설명 |
|---|---|---|
| API 주소 | `http://www.fudankw.cn:58787` | env `SKILL_SEARCH_API`로 덮어쓰기 |
| API 키 | 없음 (선택) | env `SKILL_SEARCH_KEY` |

## 자주 빠지는 함정

<Warning>
- **반드시 영어 쿼리**. 한국어/중국어로 검색하면 매칭 효과 매우 낮음
- `autonomous_safe=False` 인 스킬은 자율 행동 모드에서 실행 금지
</Warning>
