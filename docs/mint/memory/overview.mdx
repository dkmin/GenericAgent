## 언제 쓰는가
Agent가 무엇을 할 수 있고 무엇을 알고 있는지를 한눈에 파악하려 할 때 — 모든 능력의 카탈로그가 `memory/` 한 폴더에 들어 있습니다.

## 작동 원리 (Why)
GenericAgent는 atomic tool 9개 + 메모리만으로 동작합니다. 도구를 추가하지 않고도 능력을 늘리는 길은 단 하나, **메모리에 SOP/Helper를 추가하는 것**. 그래서 `memory/` 폴더는 코드가 아니라 **에이전트의 헌법, 매뉴얼, 도구상자**입니다.

## 디렉터리 구조

```
memory/
├── global_mem_insight.txt     # L1 — 인덱스 (≤30줄)
├── global_mem.txt              # L2 — 환경 사실
├── *_sop.md                    # L3 — SOP 카탈로그
├── *.py                        # L3 — Helper 모듈
├── skill_search/               # 105K 스킬 시맨틱 검색
└── L4_raw_sessions/            # L4 — 압축된 세션 로그
```
