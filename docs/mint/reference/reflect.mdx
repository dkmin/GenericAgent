## 언제 쓰는가

`launch.pyw --sched` 또는 `python agentmain.py --reflect <스크립트>`로 활성화. 메인 에이전트와 분리된 **별도 인터프리터 프로세스**에서 주기적으로 트리거를 평가합니다.

## 작동 원리

`reflect/<스크립트>.py`는 두 변수 + `check()` 함수만 정의합니다.

| 변수 / 함수 | 역할 |
|---|---|
| `INTERVAL` | 호출 주기(초) |
| `ONCE` | True면 한 번만 실행 후 종료 |
| `check()` | 매 INTERVAL마다 호출. 문자열 반환 → 메인 에이전트에 inject |

## 내장 스크립트 2개

### `reflect/autonomous.py` (5줄)

```python
INTERVAL = 1800
ONCE = False

def check():
    return "[AUTO]🤖 用户已经离开超过30分钟，作为自主智能体，请阅读自动化sop，执行自动任务。"
```

30분마다 자동 핑 → 메인 에이전트가 [`autonomous_operation_sop.md`](/memory/sops/autonomous-operation)를 읽고 자율 진입.

### `reflect/scheduler.py` (131줄)

| 동작 | 주기 |
|---|---|
| 단일 인스턴스 락 | 시작 시 `127.0.0.1:45762` bind |
| 스케줄 작업 폴링 | `INTERVAL = 120` (2분) |
| L4 아카이브 cron | 12시간 (43200초) |

```python
TASKS = '../sche_tasks/'         # 작업 정의 JSON
DONE  = '../sche_tasks/done/'    # 실행 결과 보고서
```

`_parse_cooldown`이 `repeat` 값을 timedelta로 변환:

| `repeat` | cooldown |
|---|---|
| `once` | 영구 |
| `daily`, `weekday` | 20시간 |
| `weekly` | 6일 |
| `monthly` | 27일 |
| `every_3h`, `every_30m`, `every_2d` | 임의 (정규식) |

12시간마다 자동으로 `memory/L4_raw_sessions/compress_session.py`의 `batch_process`를 호출 — `temp/model_responses/`의 라이브 로그를 압축 아카이브로 이관합니다.

## 사용 예시

<CodeGroup>
```bash 자율 모드 켜기
python agentmain.py --reflect reflect/autonomous.py
```

```bash 스케줄러 켜기
python launch.pyw --sched
# 또는
python agentmain.py --reflect reflect/scheduler.py
```

```bash 둘 다 동시
python launch.pyw --sched   # 스케줄러
# 별도 터미널
python agentmain.py --reflect reflect/autonomous.py
```
</CodeGroup>

## 새 reflect 스크립트 만들기

```python
# reflect/heartbeat.py
INTERVAL = 60
ONCE = False

def check():
    import psutil
    if psutil.cpu_percent() > 90:
        return "CPU 사용률 90% 초과 — 원인 분석하고 보고해줘"
    return None  # None이면 inject 생략
```

`check()`가 `None`을 반환하면 그 회차는 스킵. 문자열을 반환할 때만 메인 에이전트에 작업이 들어갑니다.

## 자주 빠지는 함정

<Warning>
  **포트 락 충돌**: `scheduler.py`는 45762 포트를 점유합니다. 같은 머신에서 두 번 띄우면 두 번째는 즉시 종료됩니다 (의도된 단일 인스턴스 보장).
</Warning>

<Warning>
  **`check()`가 비결정적이면 안 됩니다**: 같은 상황에서 매번 다른 문자열을 반환하면 자율 모드가 무한 루프에 빠집니다. autonomous.py가 고정 문자열인 이유입니다.
</Warning>
