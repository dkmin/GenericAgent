[`frontends/wechatapp.py`](https://github.com/dkmin/GenericAgent/blob/main/frontends/wechatapp.py) 는 iLink 봇 API 와 QR 로그인 방식을 사용해 개인 위챗 계정을 봇으로 구동합니다.

## 진입 명령

```bash
python frontends/wechatapp.py
```

`launch.pyw` 의 플래그로는 노출되어 있지 않으므로 단독 실행합니다. 이미 같은 인스턴스가 떠 있으면 `127.0.0.1:19531` 단일 인스턴스 락이 막아 종료합니다.

## 필수 설정

별도 mykey 변수는 필요 없습니다. 토큰은 첫 실행 시 QR 로그인으로 발급되어 `~/.wxbot/token.json` 에 캐시됩니다 (`TOKEN_FILE = Path.home() / '.wxbot' / 'token.json'`).

```text
~/.wxbot/token.json   # bot_token, ilink_bot_id, updates_buf 캐시
```

토큰이 없으면 자동으로 `wx_qr.png` 를 생성해 브라우저로 열어주고, 폴링으로 로그인 상태를 추적합니다.

## 주요 기능

- **QR 로그인 + 토큰 캐시** — 최초 1회 스캔, 이후 토큰 재사용.
- **롱폴링 메시지 수신** — `ilink/bot/getupdates` 로 메시지 버퍼링.
- **미디어 송수신** — `_send_media` 가 텍스트/이미지/비디오/파일을 AES 암호화 업로드 (CDN). 응답에 `[FILE:경로]` 가 있으면 자동 전송.
- **턴 단위 분할 전송** — 2000자 한도 내에서 9개 메시지까지 점진 전송, 마지막에 `[任务已完成]` 으로 마무리.
- **명령** — `/stop`, `/abort`, `/llm [N]` 으로 작업 중지·LLM 전환.
