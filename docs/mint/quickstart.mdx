<Note>예상 소요 시간: **5분**. 미리 LLM API 키 하나(Anthropic/OpenAI/Gemini 중 택일)와 Python 3.10+가 필요합니다.</Note>

## 1. 저장소 클론

```bash
git clone https://github.com/dkmin/GenericAgent.git
cd GenericAgent
pip install -r requirements.txt
```

## 2. LLM 키 설정

import MykeySetup from '/snippets/mykey-setup.mdx';

<MykeySetup />

{/* 기존 인라인 블록은 snippet으로 추출되었지만, 키 옵션 표는 그대로 유지: */}

<CodeGroup>
```python OpenAI (전체 옵션)
native_oai_config = {
    'name': 'gpt',
    'apikey': 'sk-...',
    'apibase': 'https://api.openai.com/v1',
    'model': 'gpt-5.4-mini',
    'api_mode': 'responses',
    'reasoning_effort': 'medium',
}
```

```python Anthropic (전체 옵션)
native_claude_config = {
    'name': 'claude',
    'apikey': 'sk-ant-...',
    'apibase': 'https://api.anthropic.com',
    'model': 'claude-opus-4-7[1m]',
    'thinking_type': 'adaptive',
}
```
</CodeGroup>

자세한 옵션은 [Configure LLM](/configure-llm)을 참고하세요.

## 3. UI 실행

```bash
python launch.pyw
```

기본 브라우저에 Streamlit UI가 뜹니다 (`http://localhost:185xx`).

## 4. 첫 작업 시키기

채팅 입력창에 한국어로 시켜보세요.

```text
이 디렉토리의 파일 목록을 보여주고, 가장 최근 수정된 파일 3개를 알려줘
```

처음에는 GenericAgent가 도구 호출 흐름을 만들어가는 과정을 보여줍니다. 같은 패턴의 작업을 다음에 시키면 — 메모리에 결정체화된 SOP를 바로 호출합니다.
